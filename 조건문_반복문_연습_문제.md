#  조건문 + 반복문


## 문제 1 — 숫자 여러 개 판단 

### 설명
1부터 5까지의 숫자를 차례대로 출력하면서, 각 숫자에 대해 다음을 출력하시오.

- 짝수이면 `짝수`
- 홀수이면 `홀수`

### 출력 예시 (형식은 달라도 됨)
```
1 홀수
2 짝수
3 홀수
4 짝수
5 홀수
```

### 조건
- `for`문 사용
- `if / else` 사용
- 배열 사용 ❌

```csharp
static void Main(string[] args)
{
    for (int i = 1; i <= 5; i++)
    {
        if (i % 2 == 0)
        {
            Console.WriteLine($"{i} 짝수");
        }

        else
        {
            Console.WriteLine($"{i} 홀수");
        }
    }
}
```

## 문제 2 — 조건 강화 

### 설명
1부터 10까지 출력하되,

- 3의 배수이면 `3의 배수`
- 그 외 숫자는 숫자만 출력

### 힌트
- `%` 연산자
- 조건은 하나면 충분

```csharp
static void Main(string[] args)
{
    for (int i = 1; i <= 10; i++)
    {
        if (i % 3 == 0)
        {
            Console.WriteLine($"{i} 3의 배수");
        }

        else
        {
            Console.WriteLine($"{i}");
        }
    }
}
```

## 문제 3 — 읽기 습관 훈련 

### 설명
1부터 10까지 출력하되,

- 짝수이면서 3의 배수이면 `짝수이자 3의 배수`
- 그 외는 숫자만 출력

### 힌트
- `&&` 연산자
- 조건 우선순위(괄호) 활용 가능

```csharp
static void Main(string[] args)
{
    for (int i = 1; i <= 10; i++)
    {
        if (i % 3 == 0 && i % 2 == 0)
        {
            Console.WriteLine($"{i} 짝수이자 3의 배수");
        }

        else
        {
            Console.WriteLine($"{i}");
        }
    }
}
```
