#  배열 + 반복문 

## 문제 1 — 배열에 값 넣고 출력하기 (필수)

### 설명
정수 5개를 담을 수 있는 배열을 생성하고,
1부터 5까지의 숫자를 배열에 저장한 뒤 모두 출력하시오.

### 조건
- `int[]` 배열 사용
- `for`문 사용
- 값 직접 하나씩 대입 ❌

### 힌트
- 배열 크기: 5
- 인덱스는 0부터 시작

```csharp
static void Main(string[] args)
{
    int[] ints = new int[5];
    for(int i = 0; i < 5 ; i++)
    {
        ints[i] = i + 1
        Console.WriteLine($"{i}");
    }
}
```

## 문제 2 — 배열 값 2배로 출력 (선택)

### 설명
문제 1에서 만든 배열을 사용하여
배열에 들어있는 각 값을 2배로 출력하시오.

### 출력 예시
```
2
4
6
8
10
```

---
```csharp
int[] ints = new int[5];
for (int i = 0; i < 5 ; i++)
{
    ints[i] = i + 1;

    Console.WriteLine(ints[i] *2);
}
```

## 문제 3 — 조건문 살짝 섞기 (선택)

### 설명
배열에 들어있는 값 중
- 짝수만 출력하시오.

### 힌트
- `%` 연산자
- 조건은 하나면 충분

```csharp
  int[] ints = new int[5];
 for (int i = 0; i <= ints.Length; i++)
 {
     ints[i] = i + 1;
     if (i % 2 == 0)
     {
         Console.WriteLine($"{i}");
     }

 }
```
