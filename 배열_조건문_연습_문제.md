## 문제 1 — 배열 값 중 짝수만 출력 

### 설명
정수 배열 `{ 1, 2, 3, 4, 5 }`가 있다.
이 배열에서 짝수만 출력하시오.

### 출력 예시
```
2
4
```

### 조건
- 배열 직접 초기화 가능
- `for`문 사용
- `if`문 사용

```csharp
int[] number = new int[] { 1, 2, 3, 4, 5 };

for(int i = 1; i < number.Length; i++)
{
    
    if(i % 2 == 0)
    {
        Console.WriteLine($"{i}");
    }
}
```
## 문제 2 — 배열 값 기준으로 판단 (필수)

### 설명
정수 배열 `{ 3, 7, 10, 15, 20 }`가 있다.
배열의 각 값에 대해 다음을 출력하시오.

- 10 이상이면 `크다`
- 10 미만이면 `작다`

### 출력 예시 (형식 자유)
```
3 작다
7 작다
10 크다
15 크다
20 크다
```

---
```csharp
int[] arr = new int[] { 3, 7, 10, 15, 20 };


for(int i = 0; i < arr.Length; i++)
{
    if (arr[i] >= 10)
    {
        Console.WriteLine($"{arr[i]}크다");
    }
    else if (arr[i] < 10)
    {
        Console.WriteLine($"{arr[i]}작다");
    }
```

## 문제 3 — 합계 구하기 (선택)

### 설명
정수 배열 `{ 1, 2, 3, 4, 5 }`의
모든 값을 더한 합계를 출력하시오.

### 힌트
- 합계를 저장할 변수 하나 필요

```csharp
int[] arr = new int[] { 1, 2, 3, 4, 5 };
 int sum = 0;

foreach(int i in arr) 
{
    sum += i;
}
Console.WriteLine(sum);
```

## 문제 4 — 가장 중요한 사고 훈련 (선택)

### 설명
정수 배열 `{ 4, 9, 12, 7, 20 }` 중에서
가장 큰 값 하나만 출력하시오.

### 힌트
- 비교 기준이 되는 변수 하나 필요
- 반복하면서 계속 비교

```csharp
int[] arr = new int[] { 4, 9, 12, 7, 20 };

int max = arr[0];

if (arr[0] > max)
    max = arr[1];
if (arr[2] > max)
    max = arr[2];
if (arr[3] > max)
    max = arr[3];
if (arr[4] > max)
    max = arr[4];
Console.WriteLine(max);

```
