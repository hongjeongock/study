
## 문제 1 
### 설명
정수 하나를 입력받아 다음과 같이 출력하시오.

- 10 이상이면 `크다`
- 10 미만이면 `작다`

### 조건
- `if` 와 `else`만 사용
- `Console.WriteLine`으로 출력
- 함수, 배열, 예외 처리 사용 ❌

```csharp
static void Main(string[] args)
{
    Console.WriteLine("숫자를 입력하세요:");
    int number = int.Parse(Console.ReadLine());
   
    if(number <= 10)
    {
        Console.WriteLine("크다");
    }
    else
    {
        Console.WriteLine("작다");
    }
}
```

## 문제 2 — 짝수 / 홀수
### 설명
정수 하나를 입력받아 다음과 같이 출력하시오.

- 짝수이면 `짝수`
- 홀수이면 `홀수`

### 힌트
- `%` 연산자 사용
- 조건은 하나면 충분

```csharp
static void Main(string[] args)
{
    Console.WriteLine("숫자를 입력하세요:");
    int number = int.Parse(Console.ReadLine());
   
    if(number % 2 == 0)
    {
        Console.WriteLine("짝수");
    }
    else
    {
        Console.WriteLine("홀수");
    }
}
```

## 문제 3 — 선택 문제 (안 풀어도 됨)
### 설명
정수 하나를 입력받아 다음과 같이 출력하시오.

- 0이면 `영`
- 0보다 크면 `양수`
- 0보다 작으면 `음수`

```csharp
static void Main(string[] args)
{
    Console.WriteLine("숫자를 입력하세요:");
    int number = int.Parse(Console.ReadLine());
   
    if(number > 0)
    {
        Console.WriteLine("양수");
    }
    else if(number < 0)
    {
        Console.WriteLine("음수");
    }
    else
    {
        Console.WriteLine("영");
    }
}
```
